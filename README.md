# GlobalOpt

Лендинг логистической компании **Global Opt** (доставка грузов из Китая в Россию и СНГ).

## О проекте

Проект представляет собой статический многоэкранный сайт с формами заявок, блоками преимуществ, тарифами, слайдером отзывов и встроенной картой.

## Технологии

- HTML5
- SCSS (Sass)
- CSS (собранный файл: `css/style.min.css`)
- JavaScript + jQuery
- Slick Slider (`js/slick.min.js`)
- Yandex Maps (встроенный скрипт)

## Структура проекта

- `index.html` — основная страница лендинга
- `css/style.min.css` — скомпилированные стили
- `sass/` — исходники SCSS
- `js/script.js` — инициализация слайдера
- `js/slick.min.js` — библиотека Slick
- `prepros-6.config` — конфигурация сборки для Prepros

## Как запустить

### Вариант 1: просто открыть страницу

Откройте файл `index.html` в браузере.

### Вариант 2: через локальный сервер (рекомендуется)

Пример с VS Code Live Server или любым другим static-server:

1. Откройте папку `GlobalOpt`.
2. Запустите локальный сервер.
3. Откройте адрес, который выдаст сервер (например, `http://localhost:5500`).

## Работа со стилями (SCSS)

В проекте есть исходники в папке `sass/` и файл конфигурации `prepros-6.config`.

Если вы используете Prepros:

1. Импортируйте папку проекта в Prepros.
2. Запустите компиляцию `sass/style.scss`.
3. Проверьте обновление файла `css/style.min.css`.

## Особенности

- Слайдер отзывов инициализируется в `js/script.js`.
- Для экранов меньше `400px` стрелки слайдера отключаются.
- Карта подключается через скрипт конструктора Яндекс.Карт.
